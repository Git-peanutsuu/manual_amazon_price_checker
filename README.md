# Amazon Price Tracker
Amazon商品の価格を追跡し、ユーザーのGoogleスプレッドシートに記録するWebアプリ。

## 機能
- URLからASINを抽出し、価格を記録。
- 価格変動を保存した記録からお知らせ。（上昇/下降）。
- ユーザーごとに独立したスプレッドシートを自動作成。

## 技術スタック
- Google Apps Script
- JavaScript
- HTML
- GoogleスプレッドシートAPI
